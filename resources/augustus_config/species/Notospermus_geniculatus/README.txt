# Species specific AUGUSTUS parameters for Notospermus geniculatus

Augustus parameter set name: Notospermus_geniculatus

Lineage: Eukaryota; Opisthokonta; Metazoa; Eumetazoa; Bilateria; Protostomia; Spiralia; Lophotrochozoa; Nemertea; Pilidiophora; Heteronemertea; Lineidae; Notospermus

This parameter set was created by Luo et al. (2018) using gene models created from 435 million read pairs of RNA-Seq and annotated repeats. The final parameter set was used to generate 43,294 gene models in N. geniculatus.

The resulting annotation files are available at https://marinegenomics.oist.jp/nge_v2/viewer/download?project_id=52

When using this parameter set, please cite:

Luo, YJ., Kanda, M., Koyanagi, R. et al. Nemertean and phoronid genomes reveal lophotrochozoan evolution and the origin of bilaterian heads. Nat Ecol Evol 2, 141â€“151 (2018).
